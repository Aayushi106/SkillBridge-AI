# SkillBridge AI – Design Document

## Architecture

User → Streamlit UI → Python Backend → OpenAI API → Structured Roadmap Output

## AI Component

The system uses a Large Language Model to:
- Analyze skill gaps
- Reason about career progression
- Generate structured 12-week plans
- Adapt plan based on available weekly hours

## Why AI?

Traditional roadmaps are static.
SkillBridge AI dynamically generates personalized learning paths based on user constraints and goals.

## Scalability

Future versions can include:
- Resume analysis
- GitHub profile evaluation
- Job market trend integration
- Progress tracking dashboard
